## How to Run the Website

### Method 1: Open directly

Download or clone the repository and open `index.html` in a web browser.

### Method 2: Use Visual Studio Code

1. Clone the repository:

```bash
git clone https://github.com/YOUR-GITHUB-USERNAME/Cafe-Website.git
```

2. Open the project folder:

```bash
cd Cafe-Website
code .
```

3. Install the **Live Server** extension in Visual Studio Code.

4. Right-click `index.html`.

5. Select **Open with Live Server**.

## How to Update the Website

After editing and saving the website files, run:

```bash
git add .
git commit -m "Update cafe website"
git push
```

The changes will then be uploaded to GitHub.

## GitHub Pages

To publish the website using GitHub Pages:

1. Open the repository on GitHub.
2. Go to **Settings**.
3. Select **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch.
6. Select the `/root` folder.
7. Click **Save**.

After deployment, the website will be available at:

```text
https://YOUR-GITHUB-USERNAME.github.io/Cafe-Website/
```
